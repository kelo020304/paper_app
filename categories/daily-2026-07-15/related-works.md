# 每日精选 2026-07-15 — Related Works

## 总览

WALA、Lumo-2、Xiaomi-Robotics-U0 与 See like a Robot 分别从潜在动作、预测性表征、具身数据生成和机器人系几何对齐推进具身学习。新增的 *World Models as Adversaries*（AWM）则补充了另一条相邻但重要的方法轴：不只把世界模型用于预测、规划或数据扩增，还将其条件化为可学习的对手，通过多智能体自博弈暴露规划策略的尾部风险。

前四者主要回答“如何获得更丰富、更一致、更预测性的控制数据与表征”；AWM 进一步关注“当预测模型生成的未来并非名义场景时，策略如何针对最不利但仍受交通先验约束的情形保持鲁棒”。其直接任务域是自动驾驶运动规划，而非机械臂操控；但“世界模型既是环境先验、也是训练期对手”的角色转换，为具身控制中的闭环鲁棒训练提供了可迁移的设计视角。

## 从端到端 VLA 到具备空间与未来约束的控制

RT-1、RT-2奠定了由当前视觉观测和语言指令直接预测低层机器人动作的 VLA 范式；OpenVLA、Octo 与 π0、π0.5 通过大规模视觉—语言预训练、跨机器人数据或统一生成式建模提升泛化。这类模型接口直接，但通常依赖动作标注，并默认策略能从相机图像中自行推断与动作相关的空间关系。

这一假设在相机视角变化时尤其脆弱。Universal Pose Pretraining、KYC、4D-VLA 与 SpatialVLA 向相机中心输入加入位姿、相机或空间线索；OC-VLA 则将动作重参数化到相机坐标系。See like a Robot 采取不同路线：利用深度和标定将像素反投影为末端中心化的机器人坐标系 XYZ 点图，使观测几何与动作接口从输入端共享参考系。它不要求策略隐式学习坐标变换，也不以改变动作坐标系来适配视角。

与 RoboUniView、Adapt3R 等共享三维场景表示不同，机器人中心点图不追求完整场景重建，而以图像网格承载度量 3D 坐标，因而可复用预训练 VLA 的二维视觉路径并与 RGB 逐元素融合。GeoVLA、PointVLA、3D-CAVLA、DepthVLA、Any3D-VLA 和 PointAct 通常依赖专用 3D 编码器、几何分支或点云交互；See like a Robot 的实现代价更低，但依赖可靠深度和相机—机器人标定。

动作表征是正交的另一条轴线。OpenVLA 的离散分桶，以及 FAST、Faster、RDT-2 等方法，主要处理动作 token 化、压缩与重建。Lumo-2 则以动作—世界动力学和语言语义共同塑造适合决策的潜空间；WALA 进一步利用潜在动作对齐与未来语义—几何变化，使无动作视频参与控制学习。See like a Robot、Lumo-2 与 WALA 因而并非替代关系：前者规范化策略看到的空间，后两者分别规范化内部动作表征与动作应导致的未来变化。

## 世界模型：预测、数据扩增与对抗鲁棒性

LingBot-VA、DreamZero、Motus 和 GR00T N1 从视频预测、视频生成或视频—动作联合建模出发，将控制与未来视觉演化关联；Fast-WAM 强调世界建模可主要作为训练信号，而非测试时必须显式规划。LDA-1B 则在结构化 DINO 潜空间中联合扩展动力学、策略和视觉预测。

Lumo-2 与此共享预测性动作表征目标，但不以像素级视频生成为核心：它从预训练 VLM 表征中学习面向控制的潜在物理变化，并在动作生成前显式形成未来动力学表征。相较 Lumo-1 的显式、粗到细文本推理和规划，Lumo-2 转向潜在推理，以降低 token 复杂度与推理延迟，并利用历史动作上下文消解任务阶段歧义。

WALA 与 Fast-WAM 更接近“世界模型作为训练期控制约束”的范式。其约束直接施加于 VLA 产生的潜在动作，预测 DINOv3 语义特征增量和稠密深度增量，而非像素级视频或绝对未来观测；这使无动作视频中的变化结构能够与真实动作监督共同塑造可执行的潜在动作空间。See like a Robot 不构成未来预测模型，但可为这类方法提供稳定几何参考：机器人系中的深度或点图变化更直接对应末端相对物体和场景的位移。

AWM 将世界模型的用途进一步从“预测未来”扩展到“生成最具挑战的未来”。MotionLM、Trajeglish、SMART 与 BehaviorGPT 将多智能体交通编码为离散运动 token 的自回归预测，侧重真实、交互式未来生成；DAP、Plan-R1 与 CarPlanner 等则将预测接口用于自车规划，通常围绕名义交通分布下的回报、安全或策略改进优化。AWM 同样使用离散自回归世界模型，却把原本拟合日志的被动模型条件化为可学习的稀疏对手：对手在世界模型 token 流形内产生场景条件的风险分布，规划器再以相对参考策略的 regret-CVaR 与名义性能保留约束进行优化。

这一设计也区别于 AdvSim、STRIVE、KING、Safe-Sim、DiffScene 和 LLM-Attacker 等对抗场景生成方法。后者多通过轨迹优化、外部攻击器、扩散控制或物理仿真构造关键场景；CAT、Adv-BMT、SEAL、ADV-0 等闭环鲁棒方法则将对手生成与规划器更新耦合。AWM 的特点是避免额外引入不同表征的攻击器或仿真器，而在预测世界模型自身的 token 空间中完成角色切换、反事实信用分配和联盟校准。代价是其鲁棒性、真实性与可扩展性证据目前仍主要限于文中基准和消融，尚不能直接推断到开放域机器人接触操作。

Xiaomi-Robotics-U0 则将世界模型角色前移到数据构建。相较 Dreamer、WAM、DreamZero 或 Qwen-RobotWorld 等以轨迹预测、规划或策略学习为中心的世界模型，U0 从 EMU3.5 出发，持续保留 T2I 与 X2I 共训，并将静态多视角合成、编辑式迁移和视频预测统一为离散自回归序列。它更适合作为扩增训练分布的显式生成器，而非直接定义控制潜变量或对手策略的动力学约束。

## 从无动作视频学习潜在动作

LAPA 从观测转移中学习离散潜在动作，将无动作视频转化为动作样监督；Moto 以运动 token 连接视频预训练与机器人控制；UniVLA 在 DINO 特征空间学习任务中心的潜在动作，以抑制任务无关变化。UniT 将潜在动作拓展到人类至人形机器人的迁移与世界建模，villa-X 则关注如何将其更有效整合进 VLA 预训练和控制流程。

WALA 并非仅将潜在动作用作预训练 token、运动先验或动作压缩接口。其潜在动作模型以多尺度、稀疏未来的语义特征增量和深度增量为目标；策略学习阶段冻结编码器以提供稳定对齐目标，同时保留解码器作为可训练的潜在世界模型。真实机器人动作监督持续约束潜变量的可执行性，从而统一无动作视频的动力学结构和有标注示范的控制语义。

Lumo-2 与 WALA 是互补设计：前者侧重以预测性、语义化潜空间改善 VLA 对齐和决策，后者侧重以语义—几何变化建模，使无动作视频中的物理结构迁移到真实动作控制。AWM 虽处于驾驶规划域，也说明潜在或离散世界表征还可承担第三种功能：除对齐与预测外，作为生成困难反事实的受约束训练对手。对机器人而言，这一思路只有在对抗扰动保持物理可行、任务语义不被破坏时才有意义。

## 从通用生成到具身可控观测合成

Stable Diffusion、FLUX Kontext、GPT-Image-2.0、Show-o、Janus、Janus-Pro、Chameleon、BAGEL 与 EMU3.5 将图像生成、编辑或图文统一建模扩展到较强的语义理解和指令控制；Genie、Genie2、Sora、MovieGen、Wan 与 Cosmos 进一步覆盖视频生成或世界模拟。这类基础模型擅长开放域视觉内容，却通常不直接约束机械臂姿态、跨相机几何、工作空间布局和接触交互。

Xiaomi-Robotics-U0 将可控编辑与多视角生成合并到具身观测域：以多视角深度和结构化文本控制 RGB 迁移，并将工作空间、背景、无关物、目标物与光照拆为可组合编辑维度。其目标不是生成一般三维资产，而是在机器人几何、相机视角与操作状态上保持一致，同时可针对性扰动的观测。

ViP、Rovi-Aug、Mirage 和 AnyCamVLA 通过新视角合成、数据扩增或测试时视图规范化应对相机变化；U0 也可生成多视角或扰动后的训练观测。See like a Robot 则不依赖生成式视图合成，而通过标定和深度把真实或合成图像映射到共享机器人坐标系。前者扩大视角与场景分布，后者减少同一几何状态跨视角时的表征差异；两者可结合，但共同受制于合成几何一致性、深度质量和标定精度。

## 具身视频作为合成数据与鲁棒训练引擎

Open X-Embodiment、RoboCasa、ManiSkill、BridgeData V2 与 DROID 提供真实或仿真的机器人数据基础；RoboMIND/RoboCOIN、GenieSim、InternData-A1 与 AgiBotWorld 则从具身视频、仿真或数据增强角度扩大规模。U0 将场景合成、深度条件迁移、子任务—子目标预测和多帧率视频生成纳入同一模型，形成“合成初始场景—按指令滚动生成视频—用于策略训练”的数据引擎路径。

AWM 提供了与此互补的闭环训练视角：U0 重点扩大并重组可见训练分布，而 AWM 的核心是从预测模型中主动寻找高风险、低频但仍受先验约束的交互未来，再将其用于策略更新。若迁移到具身机器人，前者可生成背景、视角、物体和光照等可组合扰动，后者可进一步按策略当前弱点选择扰动；但这需要可靠的几何一致性、接触可行性和反事实奖励归因，不能仅以视觉真实性替代物理鲁棒性。

总体而言，U0 是高保真、可组合的具身数据扩增与世界合成模型；See like a Robot 是面向预训练 VLA 的机器人系几何对齐机制；WALA 和 Lumo-2 分别将扩展观测转化为可执行潜在动作与预测性决策表征；AWM 则展示了如何把预测世界模型转化为训练期的受约束对手，以显式优化长尾风险。
