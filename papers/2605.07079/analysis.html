Learning Visual Feature-Based World Models via Residual Latent Action

价值 8/10 方法抓住了高维视觉特征生成与直接回归之间的关键矛盾，并用紧凑残差潜空间把生成建模下沉到较低维；预测、无动作模仿和模型内 RL 三组实验形成完整应用链条。主要扣分在于数据与任务规模有限、关键 RL 收益不覆盖 Panda、若干比较基线为作者适配/重实现，以及尚无独立复现。

论文提出残差潜在动作（Residual Latent Action, RLA）：把相邻视觉特征的变化压缩为紧凑潜变量，再在该低维空间用流匹配预测动态，从而兼顾特征世界模型的效率与多模态预测能力。

研究问题

现有机器人世界模型大多生成像素视频，扩散式生成计算开销大且可能产生物理不一致的幻觉；以 DINO token 等视觉特征为状态的模型虽更高效，但直接回归未来特征在复杂三维交互中易出现均值化、模糊或塌缩。直接在高维 DINO 特征空间使用扩散/流模型又很困难：论文以 512×512 输入举例，DINOv3-L token 约为百万维，远高于常见像素对齐 VAE latent 的约 1.6 万维。

核心贡献

  • 提出 RLA：编码 DINO token 残差 s_{t+h}-s_t，并结合当前特征在单次前向传播中重建未来 DINO tokens。作者实证称其具有预测充分性、跨场景泛化性和时间拓扑结构。
  • 提出 RLA-WM：不直接生成未来高维特征，而是条件于当前 DINO tokens 与动作块，在紧凑 RLA 空间中通过 flow matching 生成转移动量，再由 RLA 解码器恢复未来特征。
  • 提出面向无动作视频的极简 World Action Model：在 BC 策略上额外加一个预测 RLA 的线性头，使无动作/无本体感觉演示也能通过 RLA 监督共享视觉表征。
  • 提出 WMRL：从离线演示初始帧开始，在 RLA-WM 内进行 PPO rollout，并以与时间对齐参考视频的 DINO L1 负距离作为奖励；训练时不再与真实环境交互，也不用手工任务奖励或辅助 BC 损失。

方法原理

先用冻结的 DINOv3-L 将图像映射为 patch tokens s_t。RLA 自编码器将残差 s_{t+h}-s_t 和可学习 query 输入 Transformer 编码器，得到 z；解码器以 (s_t,z) 重建 s_{t+h}。默认 RLA 为 32 个、每个 64 维的 token，即 2048 维。RLA-WM 对 (s_t,a_{t:t+h}) 编码为条件 token；训练时对 z 与高斯噪声 ε 构造 z_τ=τz+(1-τ)ε，预测速度 z-ε；推理从噪声出发，以 30 步 Euler ODE 求解生成 z，再解码未来 DINO tokens。其生成迭代集中在紧凑 RLA token 上，而非百万维 DINO token 空间。

实验与证据

预测实验覆盖 ManiSkill（3 种机械臂、5 个任务）和真实双臂 ALOHA 的 IWS（Push T、Rope、Open Box），在 30/60 步最终帧上测 LPIPS、SSIM、DINO L1，并与 DINO-WM、直接特征扩散 RAE、直接特征流匹配 FM-WM、视频扩散 Vid2World 比较。表 1 的汇总结果中，RLA-WM 在 ManiSkill 为 LPIPS 0.071、SSIM 0.931、DINO L1 0.030，在 IWS 为 0.196、0.847、0.053；推理为 3.5T FLOPs，低于 Vid2World 的 1.1P，但高于直接回归 DINO-WM 的 2.1T。无动作视频模仿中，仅 5%（PushT 为 15%）演示带动作标签；RLA 的平均成功率 35.6%，高于 BC-ResNet 的 27.2%和 AdaWorld 的 33.7%。世界模型内 RL 的 1500-seed 评估中，RLA-WMRL 在 XArm、UR10e 上分别从 BC 的 89.9%、65.5%提升至 95.9%、73.1%，但 Panda 从 62.8%（两任务均值，按表中 84.5%、41.1%计算）降至 57.0%（74.1%、39.9%）；五项任务总平均由 59.6%升至 60.7%。以上为论文报告结果，未见独立复现证据。

局限与阅读提醒

  • RLA 从两帧的二维特征残差学习；背景运动、相机运动或工作区随机性也会占用潜变量容量，可能损害表征。作者建议转向三维、视角无关的 RLA。
  • 单帧状态与帧对训练难以处理遮挡、物体再现等部分可观测情形；模型缺乏历史记忆，作者建议扩展为多帧条件建模。
  • RLA-WM 仅预测视觉特征，不预测未来本体感觉；这限制了依赖显式运动状态的策略应用。
  • 评估集中于小规模 ManiSkill 与 IWS，且动力学模型在 ManiSkill 中按机器人、在 IWS 中按场景训练；是否可扩展到互联网规模开放世界数据尚未验证。`,`Panda`上的 WMRL 在两项拉拽任务低于 BC，作者将其归因于较高动作维度、相机遮挡、动作多样性不足和夹爪细节难以建模；这仍是尚待验证的归因。

Related Works 表述

机器人世界模型可概括为像素/视频、三维几何、特征空间及用于决策的模型内学习四条路线。像素世界模型如 Huang 等的 Vid2World 以动作条件视频扩散构建交互式世界模型，能够产生清晰画面，但 Zhang 等指出其多步预测会出现物理轨迹偏离，且计算量很高；RLA-WM 的区别是预测 DINO 特征转移的紧凑 RLA，而不是直接生成视频。Yang 等的 UniSim 同样将视频生成器用作策略训练环境，但本文称 UniSim 依赖大算力、仅报告单任务；RLA-WM 则在小规模离线视频上进行模型内 PPO。三维路线如 Yu 等的 ManiGaussian++、Huang 等的 PointWorld 和 Chai 等的 GAF 通过点云或 Gaussian/4D 表示引入空间先验，但通常带有任务或表示结构假设；RLA-WM 保持二维预训练视觉特征表征，因而避免显式三维建模，但也继承视角与遮挡局限。特征空间路线中，Assran 等的 V-JEPA 以未来特征预测服务自监督视频表征；Zhou 等的 DINO-WM 则以 DINO token 直接回归支持机器人规划。本文与二者共享“预测特征而非像素”的方向，但针对复杂三维交互中的回归均值化，改为生成紧凑的残差潜变量；它还实现了 Zheng 等 RAE 和直接特征流匹配 FM-WM 的高维生成对照，实验中报告 RLA 方案更优。潜在动作研究方面，Zheng 等的 Flare、Ye 等的 Latent Action Pretraining 与 Bu 等的 UniVLA 主要把帧间潜变量作为无动作视频模仿的代理控制；Gao 等的 AdaWorld 及 Bi 等的 Motus 则将潜在动作作为视频扩散的条件。RLA 的相对差异在于从 DINO 残差学习，并声称该变量本身足以在单步前馈中配合当前状态解码未来特征，而不只是扩散模型的弱条件；在统一 WAM 框架比较中，论文报告其优于 AdaWorld 与 UniVLA。决策层面，Hafner 等的 Dreamer、Hansen 等的 TD-MPC2 与 Li 等的 PWM 利用世界模型进行控制，但通常依赖预定义奖励、在线交互或模拟器；Lu 等的 GWM 提升策略样本效率仍需模拟器交互。本文的 WMRL 以视频对齐奖励替代手工奖励，并完全在离线视频训练出的 RLA-WM 内 rollout；“首个”此类视觉 RL 框架是作者主张，需待更广泛文献和复现确认。
