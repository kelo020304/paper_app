OpenLongTail: Generative Scaling of Long-Tail Driving Data

价值 8/10 论文把长尾自动驾驶的关键瓶颈具体化为“异构单目事件无法进入多视角策略训练管线”，并给出从位姿恢复、几何受控外推生成到闭环 VLA 验证的完整链路。相对只报生成质量的工作，其闭环数据价值验证尤其重要；但结果目前基于预印本、特定 AlpaSim/Alpamayo-R1 配置与有限 53 个事件，且真实道路安全性、不同 rig 泛化及推理成本仍待独立复现验证。

OpenLongTail 提出一个把异构、单目且无可靠位姿的长尾驾驶视频转换为目标环视相机 rig 下时空一致多视角资产的生成式数据引擎，并报告其合成数据可提升长尾场景中的闭环驾驶鲁棒性。

研究问题

端到端/VLA 自动驾驶策略依赖同步、标定的多视角日志，但动物横穿、施工区、罕见车辆等安全关键长尾事件在此类数据中稀缺且采集昂贵；大量真实世界的 dashcam、Waymo E2E 前视视频等异构素材虽包含这些事件，却缺少侧后视角、统一相机标定与可靠的米制位姿，因而不能直接用于现有多相机策略训练。

核心贡献

  • 提出 OpenLongTail：一个开源式（论文称将发布代码、模型、生成数据与转换工具）的生成数据引擎，将异构长尾驾驶视频统一为目标相机 rig 下、带位姿的同步多视角视频资产。
  • 用 MapAnything 恢复单目视频的米制自车轨迹，并以 Kalman filter 与 Rauch–Tung–Striebel 后向平滑降低位姿抖动，为后续生成建立稳定几何锚点。
  • 在 Wan2.1-VACE-1.3B 视频扩散骨干中注入逐 token 的 Plücker ray 几何；结合前视深度的同帧/跨时刻重投影，以及按相机拓扑自回归生成的跨视角记忆库，以处理侧视部分重叠与后视完全不可见区域。
  • 在 AlpaSim 的 53 个长尾事件闭环评估、未见场景的新视角生成与位姿恢复评估中验证；论文报告 NV 合成长尾数据微调后平均 AlpaSim Score 为 0.748、碰撞率 0.0%，接近使用 NV 真值多视角数据的 0.764。

方法原理

输入为 41 帧单目正前方视频与目标相机 rig。第一阶段以 MapAnything 估计每帧 SE(3) 米制位姿，再经前向 Kalman 和全片段 RTS 平滑。第二阶段以冻结 Wan VAE 和 Wan2.1-VACE DiT 1.3B 生成五个非前向相机视频，仅训练 DiT/VACE 自注意力 LoRA 及新增条件模块。几何编码器将目标相机各时空 token 的 Plücker ray、相机 ID、流角色和自车运动编码并同时加到主干、控制分支和记忆分支。深度 warp 使用冻结 DepthCrafter：cross-left/right 采用同帧前视到侧视重投影；rear-left/right/tele 借助车辆前进，从按视角设定的历史前视帧进行 lookback 重投影。随后按图 0→{1,2}、{0,1}→3、{0,2}→4、{0,3,4}→5 的拓扑顺序生成视角；记忆库含稠密像素级记忆和经 64-query Perceiver 压缩的语义记忆。训练采用 flow-matching MSE 速度预测目标，后方 tele 视角可赋更高权重。

实验与证据

生成模型在约 20 万个、41 帧的训练 clip（约 5 万场景，主要 PAV，另含 PandaSet、nuScenes）上微调；使用 32 张 H200 训练约 96 小时。未见场景新视角评估含 218 个 clip：相对 TrajectoryCrafter、Gen3C、ReCamMaster、Vista4D，OpenLongTail 在五个目标视角均有最高 PSNR、最低 LPIPS；GeoKPM 为 82.41，远高于各基线的 8.61–18.86。位姿评估覆盖 109 个场景的 218 个 clip：其米制 ATE 为 2.212，与 MapAnything 相同；相对 MapAnything，jerk 从 4737.4 降至 283.9、加速度方差从 53.22 降至 5.96。下游采用 Alpamayo-R1-10B 两阶段 SFT，并在 AlpaSim 53 个长尾事件中每事件运行两次 rollout；论文还报告加入 Waymo 合成资产可提升罕见车辆、骑行者和施工区切片，但复杂路口指标下降，归因于 Waymo 对锥桶改道和人工指挥等目标子分布覆盖不足。表 1 的 HTML 排版使各训练配方的逐行对应关系不完全清晰，以上数字采用作者正文明确陈述及可辨识的表格列。

局限与阅读提醒

  • 扩散式多视角合成存在较高推理成本。
  • 仍有残余时间伪影，尤其强外推的后向视角细节恢复更难。
  • 不同相机 rig 的相机偏差仍会保留；论文将显式相机参数条件化列为未来工作。
  • 单目中完全不可见的侧后区域本质上需要生成式补全，论文证明的是视觉/几何指标与特定仿真闭环收益，并未证明合成内容在真实道路上始终物理正确或安全。此为基于其任务设定的审慎推论。
