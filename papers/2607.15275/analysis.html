RoboTTT: Context Scaling for Robot Policies

价值 9/10 论文针对机器人策略的长上下文瓶颈给出了清晰的结构、可训练方案和真实机器人闭环证据，并同时验证尺度趋势、一次视频模仿、扰动恢复和 DAgger 在线纠错。其价值主要受限于实验任务与硬件范围较窄、试验数量有限，以及尚未证明跨骨干和跨场景泛化。

RoboTTT 将测试时训练（TTT）的可梯度更新快速权重嵌入 VLA 的动作头，以固定随时间增长的推理开销把机器人视觉—动作上下文扩展至 8K 步，并在真实双臂长时程装配中显示出更强的闭环表现、视频上下文一次模仿与在线纠错能力。

研究问题

现有机器人基础策略通常仅使用当前观测或极短历史；直接用 Transformer 保留完整轨迹会使带 KV cache 的解码延迟随上下文线性增长。短历史又不足以稳定判别多阶段装配进度、处理遮挡、从人类视频取得任务配置，或利用自身失败—纠正历史完成在线恢复。论文的目标是在不随上下文长度增加推理延迟的前提下，学习并利用数千步视觉—动作历史。

核心贡献

  • 提出 RoboTTT：在 GR00T N1.7 的 DiT 动作头每层注意力后加入 TTT 层，将跨时刻状态存入可在训练和推理时通过梯度下降更新的快速权重；注意力保留在单时刻内计算。
  • 提出长序列训练配方：对每个动作块独立采样 flow-matching 噪声的 sequence action forcing，以及跨段携带快速权重、在边界截断梯度的 TBPTT，从而在固定显存下训练最长 8K 步上下文。
  • 提出两种上下文学习用法：将人类视频仅作为更新快速权重的上下文以实现一次模仿；以及 DAgger Distillation，将机器人失败动作保留为上下文、只以人工纠正为模仿目标，蒸馏失败到纠正的映射。
  • 在三项真实双臂装配任务上报告长上下文闭环收益；随预训练上下文从 128 扩展到 8K，RoboTTT 的平均完成分数持续上升，而文中对照的 GDN 未呈现相同趋势。

方法原理

模型以预训练 GR00T N1.7 为骨干：Eagle VLM 产生视觉语言 token，DiT 输出 H 步动作块。每个时刻的寄存器、关节状态和加噪动作 token 先在时刻内自注意力/交叉注意力；随后沿时间维送入 TTT 层。TTT 的快速模型是两层 GeLU MLP：对当前 token 的 key/value 投影，以 MSE 自监督损失更新快速权重 W，再用更新后的 W 查询 query，故历史被压缩在权重空间而无需保存全部 KV。每层以近零初始化的 tanh 门控将 TTT 输出加回预训练注意力输出，减弱对原有能力的初始扰动。训练时，外层使用逐步 flow-matching 动作损失，内层执行快速权重更新；W 的初值、Q/K/V 投影及其更新动力学由外层任务损失学习。TBPTT 只在短片段内反传，但快速权重跨片段连续传递且在边界 detach。预训练阶段仅训练新增 TTT/GDN 序列层，任务后训练再微调全模型。部署时，每条 rollout 从学习到的 W0 出发，每时刻更新一次快速权重并以 k 步去噪生成动作块。

实验与证据

真实机器人实验使用四相机 YAM 双臂平台，控制频率 30 Hz；任务为 Pup Go Car（约 2 分钟）、Circuit（约 1 分钟、80 个配置，20 训/60 测）和 Gear Bot（约 5 分钟、十阶段）。主实验中 RoboTTT 的平均 rubric 完成分数为 79%，论文称相对单步 GR00T N1.7 的 42% 提升 87%，相对最佳基线 GDN 的 56% 提升 41%；完全成功次数分别为 9/20、13/20、2/10，是唯一完成 Gear Bot 的方法。上下文尺度实验中，8K 预训练的平均分为 71.5%，相对同模型 1K 的 43.9% 高 63%，并高于最佳短上下文基线 45.6% 的 57%；摘要写作“62%”，与正文的“63%”存在一处数值表述不一致，应以正文图表对应描述为准。一次视频模仿中，在未见 Circuit 配置上，RoboTTT 成功 6/10、完成分数 65%，GDN 为 0/10、33%。扰动恢复中，屋顶被移除后 RoboTTT 为 15/20，优于 GDN 13/20 和短上下文基线至多 10/20；轮胎扰动中 RoboTTT 与 GDN 均为 18/20。对 100 条 DAgger 轨迹，标准 DAgger 对四种方法平均提升 9%，而 DAgger Distillation 对两种序列模型平均提升 33%（RoboTTT 36%、GDN 29%）。消融显示：去除 sequence action forcing 显著降低闭环表现；线性快速模型优于 GR00T 基线但比 MLP 版本低 27%；加入历史动作 token 和寄存器 token 分别带来 23% 与进一步 18% 的相对提升。

局限与阅读提醒

  • 8K 上下文扩展的是训练与状态更新能力，但论文仅在三种桌面双臂装配任务、单一硬件设置与有限试验次数（多数 20 次，Gear Bot/视频模仿 10 次）上验证，泛化范围尚未充分建立。
  • 长上下文训练增加训练成本；作者明确将更高效的 TTT 训练（如 TNT）留作未来工作。
  • TTT 内层采用通用 key-value MSE 更新目标；作者指出仍可探索面向机器人的 TTT 目标。
  • 尽管性能提高，模型仍不能处理部署中的所有失败模式；作者建议结合直接优化任务成功率的强化学习。论文没有给出安全性、灾难性错误率或跨机器人形态迁移的系统评估。

Related Works 表述

长上下文机器人策略方面，RT-1（Brohan 等）、RT-2（Brohan 等）、OpenVLA（Kim 等）、π0（Black 等）、GR00T N1（NVIDIA 等）及部分 WAM 通常以当前帧为主；RDT-1B（Liu 等）、Octo（Octo Model Team）、SmolVLA（Shukor 等）等可使用少量连续帧，但论文概括其窗口通常仅 2–8 帧。TraceVLA（Zheng 等）通过视觉轨迹提示扩展时序线索，Past-Token Prediction（Torne 等）预测过去动作，ContextVLA（Jang 等）压缩多帧上下文，Gated Memory Policy（Gao 等）缓存并门控历史，均旨在利用历史，却仍是固定窗口或固定记忆设计。BPP（Mark 等）以关键历史帧、MEM（Torne 等）以多尺度具身记忆，以及以关键帧或语言作高层记忆的工作，选择性地摘要长历史；RoboTTT 的差异是直接处理连续视觉—动作流，并以快速权重学习保留/遗忘，而非显式检索或预设摘要。Decision Transformer 类的整段自回归历史模型及相关长上下文序列策略能够建模远程依赖，但论文指出其 KV-cache 解码延迟随历史增长，不适合长时程实时部署；传统 LSTM 等 RNN 虽有常数推理复杂度，却在大规模建模上通常不及全注意力。RoboTTT 将快速权重视作 RNN 状态，在状态容量和更新规则上区别于普通向量隐状态；其 GDN 对照也显示，仅采用线性复杂度递归记忆并未随预训练上下文增长获益。关于历史引发的 causal confusion/copycat 问题，de Haan 等与 Wen 等指出策略可能依赖过去观测中隐含的自身动作；既有工作以历史摘要、辅助目标或选择性跳过上下文缓解。RoboTTT 的主张是经任务外层损失训练的快速权重可动态编码相关信息、丢弃冗余特征；这是一项实验支持的机制解释，而非本文给出的形式化保证。 测试时训练方面，Sun 等提出利用自监督目标在测试时更新模型以应对分布变化；Sun 等的 TTT/“Learning to (Learn at Test Time)”把快速权重作为表达力更强的递归状态，Titans（Behrouz 等）等将测试时记忆用于长序列，TTT3R（Chen 等）和视频生成工作将其推广至连续视觉任务。RoboTTT 沿用“训练和推理均更新快速权重、将上下文压缩到参数空间”的核心范式，但将其嵌入 VLA 的 flow-matching DiT 动作头，使用机器人动作流作为序列，并针对超长轨迹配合 sequence action forcing 与 TBPTT。Evolve-VLA（Bai 等）、On-the-fly VLA adaptation（Liu 等）、TTT-Parkour（Zhu 等）等也使用“test-time training”表述，但按本文区分，它们在测试任务额外收集数据并微调整体模型，并非持续更新小型快速权重；VITA（Ziakas 与 Russo）是更接近的机器人设定，使用快速权重适配 VLM 价值函数，而 RoboTTT 直接构建视觉运动控制策略。本文还将 Algorithm Distillation（Laskin 等）的“把学习过程蒸馏进序列模型”思想具体化为 DAgger Distillation：失败动作作为上下文、人工纠正作为目标；区别于标准 DAgger（Ross 等）通常只用纠正数据拟合策略。 机器人基础模型方面，VIMA（Jiang 等）、RT-2、OpenVLA、π0、RDT-1B、GR00T 等的共同范式是从视觉语言预训练表示出发再接动作生成模块。动作表示上，RT-2 等采用离散动作 token 的自回归预测，π0、RDT-1B 与 GR00T N1.7 等采用扩散/flow-matching 连续动作头。RoboTTT 宣称架构上可作为多种机器人基础模型的可插拔序列模块，但本文只在 flow-matching 的 GR00T N1.7 上实例化；因此，对其他离散动作或不同 VLA 骨干的可迁移性尚未由实验验证。相对这些工作，论文最直接的新增证据是：在其设置中，预训练上下文由 128 扩至 8K 时，闭环指标稳定提升，且 8K 对应约 30 Hz 下超过四分钟的历史。
